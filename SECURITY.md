# Security Policy

## Supported Versions

We currently support the following versions of this project with security updates:

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a potential security vulnerability, please report it immediately to [security@metyatech.io](mailto:security@metyatech.io).

Please provide as much information as possible, including:

- Steps to reproduce the vulnerability.
- Potential impact.
- Any suggested mitigations.

We take security seriously and will respond as quickly as possible to evaluate and address any reported issues.

Do not disclose the vulnerability publicly until we have had a chance to investigate and provide a fix.
